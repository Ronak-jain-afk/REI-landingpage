# RE-Invoice Landing Page

The official landing page for **RE-Invoice**, a robust, offline-first open-source application for inventory management, billing, and invoice generation.

## Tech Stack

- **Framework**: [Next.js 16 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS 4](https://tailwindcss.com/)
- **UI Components**: [shadcn/ui](https://ui.shadcn.com/) (Base UI)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Deployment**: [Vercel](https://vercel.com)

## Features

- **Dynamic Versioning**: Automatically fetches the latest release details and download links for Windows (.exe) and Linux (.deb) from the GitHub API.
- **Modern Aesthetic**: High-performance "Modern Fluidity" design with animated mesh gradients and responsive layouts.
- **Dark Mode**: Native support for system preferences and manual theme switching.
- **Performance Optimized**: Fully static generation with Next.js image optimization for fast load times across all devices.

## Local Development

1. Navigate to the landing page directory:
   ```bash
   cd Landing-page
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
